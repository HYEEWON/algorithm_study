### 1991. 트리 순회 (Silver1, Python)
* 틀린 이유: 트리를 구현했는데 트리 노드를 추가하는 부분에서 객체가 아닌 str이 넘어가서 에러 발생
* 수정: 순환을 하며 객체를 추가하지 않고, 그냥 객체를 여러 개 생성
